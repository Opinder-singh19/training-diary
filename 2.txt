Again Some tags 
       <hr> Tag (Horizontal Rule):
	•	      	           •	Used to draw a horizontal line across the web page.
	•	             	•	It is mainly used to separate sections or content.
	•	            	•	It is an empty tag (no closing tag needed).

         <marquee> Tag:
	•	               It is used to make text or images move (scroll) on the web page.
	•	              The content can scroll left, right, up, or down.
	•	              It is a non-standard tag, so it's not recommended in modern websites (but still works in most browsers).

List tags :->
There are three list tags :->
1) ordered   2) unordered  3) description list 
—> <ol> Tag (Ordered List):                                     —> unordered list
                                                                                                <ul> Tag (Unordered List):
          Used to create a numbered list (1, 2, 3...).                                       Used to create a bulleted list (•)
	•	                                 Each item in the list goes inside the <li> tag.                              Each item is written inside an <li> tag.

	•	         Description List (<dl>, <dt>, <dd>)
	•	         Used to show terms and their descriptions — like a glossary or FAQ.
	•	                                       Tags:
	•	               <dl> – Description List (the container)                 
	•	               <dt> – Description Term (like a title or word)
	•	               <dd> – Description Data (explains the term)
