WHAT IS HTML ?

1) HTML stands for HyperText Markup language
2)It is used to create and design web pages.
3) HTML tells the browser what to show (text, images, links, etc.).
4) It uses tags like h1, p, img to structure the page.
5) HTML is the skeleton of every website.

WHAT ARE THE USES OF HTML?

Creates and designs web pages
Adds text, images, links, and videos to a website.
Helps make forms for login, signup, etc.
